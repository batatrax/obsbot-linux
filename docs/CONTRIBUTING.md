# Contributing to obsbot-linux

Thank you for your interest! Here is how to contribute.

## Development Setup

1. Fork the repository
2. Clone your fork
3. Set up the SDK (see README)
4. Create a feature branch: `git checkout -b feature/my-feature`
5. Make your changes
6. Test on both obsbot-linux and obsbot-kde
7. Submit a Pull Request

## Code Style
- C++17, Qt6 conventions
- French UI strings (English code comments)
- Use the shared `obsbot-core` library for SDK calls
- Add widgets to both apps when possible

## Issues
Please use GitHub Issues for bug reports and feature requests.
